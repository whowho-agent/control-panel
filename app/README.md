# FastAPI control-plane skeleton

Pragmatic clean architecture layout:

- `app/domain`
- `app/repos`
- `app/services`
- `app/api`
- `app/templates`

Current scope:
- dashboard UI via FastAPI templates
- clients UI via FastAPI templates
- list/create/delete frontend clients
- topology health
- frontend config API
- relay config API

Entry point:
- `app.main:app`
